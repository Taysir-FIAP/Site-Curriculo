# 📄 Currículo Profissional - Checkpoint 1

## 📌 Sobre o Projeto
Este repositório contém a primeira etapa da **CP01** da disciplina de *Application Development* da FIAP. O objetivo foi desenvolver um currículo web estruturado, utilizando boas práticas de **HTML5** e **CSS3**, com foco em semântica e estilização avançada sem o uso de frameworks externos.

O projeto utiliza uma arquitetura de navegação baseada no pseudo-seletor `:target`, permitindo a alternância entre seções (Sobre, Formação, Experiência) em uma única página, mantendo a performance e a fluidez visual.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica de dados e conteúdo.
- **CSS3**: Estilização personalizada, Flexbox para alinhamento e lógica de abas dinâmicas.
- **Versionamento**: Git e GitHub para controle de alterações e gerenciamento de branches.
